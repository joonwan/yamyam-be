## RAG 시스템 활용 계획

### 비만 환자 프로필 분석
사용자 프로필 정보를 기반으로 다음을 계산:
- **BMI**: 체중(kg) / 신장(m)²
- **비만 기준** (한국 기준):
  - 정상: 18.5 ≤ BMI < 23
  - 과체중: 23 ≤ BMI < 25
  - 비만: BMI ≥ 25
  - 고도비만: BMI ≥ 30

### 체중 감량 목표 설정
- 현재 체중의 5-10% 감량을 초기 목표로 설정
- 주당 0.5-1kg 감량 권장
- 급격한 체중 감량 경고

### 식단 분석 시 체크 항목
- ✅ 총 칼로리 섭취량 (목표 칼로리 대비)
- ✅ 탄수화물/단백질/지방 비율
- ✅ 포만감 지수 (식이섬유 함량)
- ✅ 고열량 저영양 식품 섭취 빈도
- ✅ 간식 및 야식 패턴

### 권장 식단 패턴
- 저칼로리 균형식 (하루 -500~-1000kcal)
- 고단백 식단 (포만감 증가, 근육량 유지)
- 저GI(Glycemic Index) 식품 선택
- 충분한 채소 섭취 (식이섬유)

### Vector DB 적재 대상
- 비만 환자 권장 식품 리스트
- 체중 감량에 도움되는 식사 패턴
- 피해야 할 고열량 음식 리스트
- BMI 구간별 목표 칼로리 계산식

### 프롬프트 엔지니어링
```
당신은 대한비만학회 인증 비만 전문 영양사입니다.
사용자의 BMI는 {BMI}이며, {비만_단계}에 해당합니다.
체중 감량 목표는 현재 체중 {current_weight}kg의 {target_percentage}%인 {target_weight}kg입니다.

다음 사항을 고려하여 식단을 분석하세요:
1. 목표 칼로리: {target_calories}kcal/일
2. 탄수화물:단백질:지방 = 50:25:25 권장
3. 식이섬유 25g 이상 섭취 권장
4. 나트륨 2,300mg 미만 제한
5. 고열량 저영양 식품 섭취 최소화
```

## 관련 BMI 기준 (한국 성인)

| 분류 | BMI 범위 |
|------|----------|
| 저체중 | < 18.5 |
| 정상 | 18.5 - 22.9 |
| 과체중 | 23.0 - 24.9 |
| 1단계 비만 | 25.0 - 29.9 |
| 2단계 비만 | 30.0 - 34.9 |
| 3단계 비만 (고도비만) | ≥ 35.0 |
