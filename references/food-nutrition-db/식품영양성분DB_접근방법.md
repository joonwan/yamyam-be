# 식품 영양성분 데이터베이스 접근 방법

## 1. 전국통합식품영양성분정보 표준데이터

**출처**: 식품의약품안전처
**URL**: https://www.data.go.kr/data/15100064/standard.do

### 데이터 개요
식품의약품안전처, 농촌진흥청, 해양수산부에서 제공하는 원재료성 식품, 가공식품의 영양성분 정보를 통합한 표준데이터

### 주요 데이터 항목 (약 40개)
- 식품코드, 식품명
- 에너지(kcal)
- 3대 영양소: 단백질, 지방, 탄수화물
- 당류, 식이섬유
- 무기질: 칼슘, 철, 인, 칼륨, 나트륨
- 비타민: A, B1, B2, C, D
- 콜레스테롤, 포화지방산

### 파일 포맷
XLS, XML, JSON, RDF, CSV

### 다운로드 방법
- **그리드 데이터**: PC버전에서 5만 건 제한
- **전체 데이터**: Open API 활용 권장
- **갱신주기**: 연간 (최근 수정일: 2025-06-30)

## 2. 전국통합식품영양성분정보(음식) 표준데이터

**출처**: 식품의약품안전처
**URL**: https://www.data.go.kr/data/15100070/standard.do

### 데이터 개요
국민건강영양조사, 식품영양성분통합데이터베이스, 국가표준식품성분표를 기반으로 한 **음식(조리된 식품)**의 영양성분 정보

### 추가 데이터 항목
원재료 데이터의 모든 항목 + 다음 항목:
- 데이터구분, 식품기원
- 식품분류(대/중/소/세분류)
- 1인분량 참고량, 식품중량
- 데이터생성방법, 생성일자
- 출처정보

### 파일 포맷
XLS, XML, JSON, RDF, CSV

### RAG 시스템 활용 계획
이 데이터는 사용자가 입력한 **식단(음식명)**과 매칭하는 데 매우 유용합니다.

## 3. 국가표준식품성분 데이터베이스 10.3

**출처**: 농촌진흥청
**공식 사이트**: http://koreanfood.rda.go.kr
**안내 URL**: https://www.rda.go.kr/board/boardfarminfo.do?mode=view&prgId=day_farmprmninfoEntry&dataNo=100000802611

### 데이터 규모
- **식품 수**: 3,330점
- **영양성분 종류**: 130종
- **데이터 건수**: 약 29만 건
- **데이터 결측률**: 32% (2025년 기준, 이전 41%에서 개선)

### 최신 업데이트 (10.3)
- 신규 식품 84점 추가 (가공 닭가슴살, 칼슘 강화 쌀, 마시멜로 등)
- 101점 식품 영양정보 최신화

### 다운로드 방법
1. 농식품올바로 누리집(koreanfood.rda.go.kr) 접속
2. 공지사항에서 "국가표준식품성분 DB 10.3 공개 및 다운로드 안내" 확인
3. 제공 포맷: hwp, hwpx, pdf

### 웹사이트 기능
- 식품명 검색
- 영양가 계산 기능

### 활용처
- 질병관리청 국민건강통계
- 교육부 학교 급식 시스템
- 건강관리 앱 개발

## 4. 식품안전나라 - 나트륨·당류 정보

**출처**: 식품의약품안전처 식품안전나라
**URL**: https://foodsafetykorea.go.kr

### 제공 정보
- 나트륨 정보
- 당 정보
- 삼삼한 밥상 (저염식 레시피)
- 미각 판정 도구

### 접근 방법
메인 사이트 > 건강·영양 > 나트륨ㆍ당류 줄이기 섹션

## RAG 시스템 구축 시 권장 사항

### 우선순위 1: 전국통합식품영양성분정보(음식)
- 사용자가 입력하는 식단은 대부분 조리된 음식
- API를 통해 전체 데이터 수집 필요
- Vector Store에 적재하여 유사 음식 검색에 활용

### 우선순위 2: 국가표준식품성분 DB
- 가장 상세한 영양성분 정보 (130종)
- 신뢰도 높은 공공 데이터
- 원재료 단위 분석에 유용

### 우선순위 3: 전국통합식품영양성분정보(원재료)
- 가공식품 및 원재료 분석
- 식단 구성 시 대체 식품 추천에 활용

### 데이터 통합 전략
1. 공공데이터포털 Open API 활용 (API 키 발급 필요)
2. koreanfood.rda.go.kr에서 Excel/CSV 다운로드
3. 중복 데이터는 식품코드 기준으로 병합
4. Redis Vector Store에 임베딩하여 검색 최적화
