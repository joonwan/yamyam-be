## RAG 시스템 활용 계획

### Vector DB 적재 대상
- 3개 주요 PDF에서 연령/성별/활동량별 섭취기준 표 추출
- 에너지 적정 비율 데이터
- 질환별 영양소 권장/제한 사항

### 프롬프트 엔지니어링
KDRIs 자료를 기반으로 Gemini 2.0 Flash에게 다음과 같은 컨텍스트 제공:
- "당신은 대한민국 보건복지부 인증 영양사입니다"
- "2020 한국인 영양소 섭취기준을 준수하여 분석합니다"
- "사용자의 나이, 성별, 활동량을 고려한 개인 맞춤 권장량을 제시합니다"

### 주요 데이터 포인트
1. **에너지 적정 비율** (6세 이상)
   - 탄수화물: 55~65%
   - 단백질: 7~20%
   - 지질: 15~30%
   - 포화지방산: <7~8%
   - 트랜스지방산: <1%

2. **콜레스테롤** (19세 이상)
   - 권고: <300mg/일

3. **나트륨 만성질환위험감소 섭취량**
   - 성인: 2,300mg/일 미만
