## RAG 시스템 활용 계획

### 당뇨병 환자 식단 분석 시 체크 항목
- ✅ 당류 함량 (설탕, 꿀, 과당 등)
- ✅ 식이섬유 함량
- ✅ 식사 시간 규칙성
- ✅ 지방산 구성비 (포화/불포화)
- ✅ 나트륨 함량

### 고혈압 환자 식단 분석 시 체크 항목
- ✅ 나트륨 함량 (1일 2,300mg 미만 권장)
- ✅ 칼륨 섭취 (혈압 조절 도움)
- ✅ 포화지방산 제한
- ✅ 음주 여부

### Vector DB 적재 대상
- 각 질환별 금지 식품 리스트
- 권장 식품 리스트
- 영양소 섭취 제한 기준
- 식사 패턴 권장사항

### 프롬프트 엔지니어링
사용자 프로필에 질환 정보가 있는 경우:
```
당신은 당뇨병/고혈압 전문 영양사입니다.
사용자는 {질환명} 환자이므로, 다음 사항에 특히 주의하여 분석하세요:
- 당뇨병: 당류 제한, 식이섬유 증가, 혈당지수 고려
- 고혈압: 나트륨 제한(<2,300mg/일), 칼륨 증가, DASH 식단 권장
```
