# 당뇨병 식사요법 핵심 정리

출처: 대한당뇨병학회
URL: https://www.diabetes.or.kr/general/dietary/dietary_01.php

## 식사요법의 목표

"고혈당, 고지혈증 등의 대사이상을 교정하여 합병증을 예방하고, 좋은 영양 상태를 유지"하는 것입니다.

## 식사요법 실천의 주요 효과

- 혈당과 혈중 지질 농도를 정상 수준으로 유지
- 적절한 체중 관리를 통한 인슐린 저항성 개선
- 합병증 예방 및 치료 지원
- 영양 불균형 상태 예방

## 식사요법의 6가지 기본원칙

| 원칙 | 내용 |
|------|------|
| **규칙적 식사** | 일정한 시간에 정해진 양의 음식 섭취로 혈당 조절 |
| **당류 제한** | "설탕이나 꿀 등 당류의 섭취를 주의"—급격한 혈당상승 유발 |
| **식이섬유** | 완만한 혈당 상승과 혈중지방 농도 조절 |
| **지방 관리** | 포화지방산·트랜스지방산 줄이고 불포화지방산 선택 |
| **저염식** | 과다 나트륨 섭취 제한으로 혈압 관리 |
| **금주** | 술은 영양가 없이 높은 에너지 함유 |

## 핵심 메시지

당뇨식사요법은 제한이 아닌 "건강한 식사를 계획하고 실천"하는 긍정적 접근입니다.

## RAG 시스템 활용 방안

- 사용자의 식단 분석 시 당류 함량 체크
- 규칙적인 식사 시간 패턴 분석
- 식이섬유 섭취량 계산 및 권장
- 지방 구성비 (포화/불포화) 분석
- 나트륨 섭취량 모니터링
